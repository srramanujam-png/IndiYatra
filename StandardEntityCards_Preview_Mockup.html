Standard Entity Cards + Preview Popup
One consistent card pattern for Course, Theme, Module, Lesson and Snippet. Tapping a card's body opens a preview popup — image, category, title, description, and a ✕ to close (which returns you to exactly where you were, since this never navigates away — it's an overlay on top of the same screen). Every card now also carries a like and bookmark icon (each with a count, both toggle and turn saffron when active), a play button that fires the same action the row's own CTA used to, and — for Course / Theme / Module only — a colour-coded progress dial.
